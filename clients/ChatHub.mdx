title: "在 ChatHub 中使用"
description: "1. 通过 ChatHub 官网安装 Chrome 插件：https://chathub.gg"
---

1. 通过 ChatHub 官网安装 Chrome 插件：[https://chathub.gg](https://chathub.gg)

2. 点击 ChatHub 插件左下角的齿轮进行设置（图中红1）

3. 点击“通过 API key 使用”列表中 OpenAI 的“配置”按钮（图中红2）

4. 输入你的 AiHubMix API key，在[本站的Key](https://aihubmix.com/token)中生成（ key 的格式为：sk-\*\*\*），之后复制填入（图中红3）

5. API Host 改成 AiHubMix 网站的地址：（图中红4）

```
https://aihubmix.com
```
