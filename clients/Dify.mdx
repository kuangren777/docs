#### 4. 模型配置

根据业务需求调整参数（如提示词、分辨率、宽高比等）。

#### 5. 配置完成后运行

工作流运行成功后，即可调用 CaMeL AI 生图 API 生成图片。

<Tip>
  接口返回**URL** 和 **Base64** 两种图片格式。 在实际使用中请根据场景进行格式转换。
</Tip>
