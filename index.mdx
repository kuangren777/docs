## 为什么选择 CaMeL AI？

* **独创智能缓存分组技术**：哪怕你用的不是官方 Claude Code，而是第三方开源工具/IDE，我们也能通过算法大幅提升缓存命中率。命中缓存后，Opus 4.6 等顶级模型一次调用可能只要几分钱。
* **拒绝低智模型**：只做正版高质量模型转发，一次跑通省下的时间远比低价渠道反复 Debug 划算。
* **科研级信誉**：多家科研院所、高校实验室稳定使用，支持开具正规发票（普票/专票），支持对公。
* **灵活计费**：无会员、无包月，按量付费，计费规则与官方对齐。

## 分组说明

| 分组 | 适用场景 | 说明 |
|------|---------|------|
| **default** | Claude Code 等自带缓存管理的应用 | 高可用组，满血 Max 渠道 |
| **smartcache** | Cherry Studio / OpenClaw 等不自带缓存管理的应用 | 独创智能缓存分组，大幅提升命中率 |
